Project: Simple Calculator

A functional calculator for basic arithmetic with enhanced features like keyboard input and memory (M+, M−, MR, MC). Flip on Scientific mode for functions like sin, cos, tan, log, ln, √, xy, and constants π and e. Includes a light/dark theme switcher.

  • Basic: digits 0–9, + − × ÷, %, ., clear, backspace, equals.
  • Enhanced: keyboard input, memory (M+, M−, MR, MC).
  • Advanced: scientific mode (sin, cos, tan, log, ln, √, ^, π, e), theme switcher.
